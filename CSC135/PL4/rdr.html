RDR

Grammar

EXP         ::= EXP + TERM | EXP - TERM | TERM
TERM      ::= TERM * FACTOR | TERM / FACTOR | FACTOR
FACTOR  ::= ( EXP ) | DIGIT
DIGIT       ::= 0 | 1 | 2 | 3

Intructions

This program is to check if a string is a valid math expression using '(' ')' '+' '-' '*' '/' '0' '1' '2' 3' characters where the string must ends with '$'
Therefore, in order to get a correct result, you must type in a right expression similar to the valid examples belows.

Example

Valid set: 1+2$ , 0-3*2$, 2/(3+1)$
invalid set: 1+2, 1*4$, 1*a$
